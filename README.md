## ANGULAR QUOTE
## By FELIX KIBET KURGAT*
## Description:
	*It shows the quote
	*It shows the day the quote was created
	*It shows how long the quote since it was posted
	*It shows the authors name
## The aim of this is
	*To advertise the quotes
## Setup/Installation Requirements
This site is hosted in gh-pages
